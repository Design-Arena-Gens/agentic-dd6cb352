# Menopause Expedition eBook

A Next.js site delivering a humorous, tour-guide-style ebook that supports women navigating menopause without diet or holistic medicine directives, while offering guidance on intimate relationships.

## Scripts

- `npm run dev` – start local development
- `npm run build` – create a production build
- `npm run start` – run the production server

Deployable to Vercel as `agentic-dd6cb352`.
